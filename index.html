My Projects & Skills

Semi-Automated Irrigation System

This project aims to optimize water usage in agriculture by combining automation and manual intervention. Using sensors to monitor soil moisture, temperature, and humidity, the system automates the irrigation process. The farmer can control irrigation remotely or manually override the system as needed. The goal is to conserve water, reduce labor, and improve crop yield.

IoT-based pulse monitor

This project focuses on healthcare innovation by creating a pulse rate monitoring device using IoT. The system captures real-time heart rate data through a wearable device or sensor module and transmits it to a connected platform, such as a smartphone or cloud database. It is designed for easy use by individuals for health monitoring and can provide alerts in case of abnormal readings.

Symptom checker

The symptom checker website is a user-friendly platform designed to assist users in identifying potential health issues based on their symptoms. Users can input symptoms, and the website provides possible conditions using a database of medical knowledge or AI algorithms. This tool promotes early health awareness and encourages users to consult healthcare professionals when necessary.
